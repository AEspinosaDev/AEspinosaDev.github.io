Limo-More

Limo-More is an arcade driving/puzzle game. The limousine will get longer and longer every second, so move quickly and collect the 3 orbs before it is too late.

This game was made for the spanish Game Jam called Tu juego a juicio Jam. It was made using Unity 2020.3.16f1. The theme of the Game Jam was: Too long.

The jam lasted 2 weeks, but, due to personal reasons, the game was fully developed in 4 days.
